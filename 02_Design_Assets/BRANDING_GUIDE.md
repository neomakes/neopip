# 🎨 BRANDING GUIDE: PIP 디자인 시스템 상세 매뉴얼 v2

이 문서는 Figma의 디자인 시스템을 앱 코드 (`03_Development`)로 구현하기 위한 최종 가이드라인을 제공합니다.

---

## 1. 브랜드 철학 및 UX/UI 원칙 (Brand Philosophy & UX/UI Principles)

이 원칙들은 PIP의 모든 시각적, 상호작용적 경험을 결정하는 최상위 가이드라인입니다.

### 1.1. 브랜드 페르소나: 이타적인 딥테크 기술자 (Altruistic Deep-Tech Technician)
PIP는 '팔란티어'처럼 냉철하게 데이터를 분석하지만, '스포티파이'처럼 따뜻하고 친절하게 정보를 제공하며, '로버트 기요사키'처럼 지적인 선배로서 사용자의 성장을 돕는 동반자입니다.
*   **정의:** **"인간 중심의 지능 (Human-Centered Intelligence)"**.

### 1.2. 핵심 UX 원칙: 차분한 명료함 (Calm Clarity)
사용자는 1분 내외의 짧은 감정 기록(저널링)을 통해 자신의 내면을 들여다보고, 통제감과 평온함을 느끼며 스스로를 더 깊이 이해하며 성장하는 경험(Self-realization)을 제공받습니다.

### 1.3. 데이터 시각화 철학: 예측적 인사이트 (Glanceable & Predictive Insights)
PIP의 데이터는 '오늘의 마음 날씨 예보'처럼, 복잡한 수치 대신 그날 예측되는 상태를 단순하고 직관적인 지표로 제안합니다. 이는 군더더기 없는 사실을 전달하고, 사용자가 그 사실을 바탕으로 스스로의 감정과 해석을 덧붙일 수 있는 성찰의 공간을 제공하기 위함입니다.

---

## 2. 시각적 메타포: Glass Gems & Orbs

PIP의 핵심 데이터는 '유리 질감의 보석(Gems) 또는 구체(Orbs)'라는 메타포를 통해 표현됩니다. 이는 데이터의 여러 차원을 직관적으로 시각화하기 위함입니다.

*   **밝기 (Brightness):** Gem/Orb의 밝고 어두움은 **데이터 수집의 완성도**를 나타냅니다. 데이터가 충분히 모일수록 더 밝게 빛납니다.
*   **기하학적 형태 (Geometry):** Gem/Orb의 다양한 기하학적 형태는 **데이터의 고유한 특성(감정, 신체, 행동 등)**을 반영합니다.
*   **네온 그림자 (Neon Shadow):** Gem/Orb를 둘러싼 네온 스타일의 그림자는 AI 모델이 해당 데이터 분석에 대해 갖는 **'불확실성(Uncertainty)'**을 표현합니다. 확신이 높을수록 그림자는 옅어집니다.

## 3. 컬러 팔레트 (Color Palette)

### 3.1. 배경 (Background)
앱의 전반적인 깊이감과 차분함을 위해 동적인 그라데이션 배경을 사용합니다.
*   **방식:** 선형 그라데이션 (Linear Gradient)
*   **색상:** 상단 `#000000` 에서 하단 `#202020` 으로 점진적으로 어두워지는 방식.

### 3.2. 주조색 (Primary & Accent: Teal System)
앱의 로고 및 핵심 상호작용에 사용되는 Teal 계열 색상입니다.
| 역할 | Hex Code |
| :--- | :--- |
| **Teal/Brightest** | `#82EBEB` |
| **Teal/Bright**    | `#40DBDB` |
| **Teal/Default**   | `#31B0B0` |
| **Teal/Medium**    | `#248888` |
| **Teal/Dark**      | `#176161` |
| **Teal/Darkest**   | `#0B3D3D` |

### 3.3. 중립색 (Neutrals)
텍스트와 기본 아이콘 등은 대부분 흰색을 사용하여, 다채로운 Gem/Orb와 명확히 대비되도록 합니다.
*   **Primary Text & Icons:** `#FFFFFF` (White)
*   **Secondary Text:** `#A0A0A0` (Light Gray) - 보조 정보에 사용

## 4. 아이코노그래피 (Iconography)

*   **스타일:** MVP에서는 미니멀리즘을 극대화하기 위해, 대부분 단색(Monochrome)의 **흰색(` #FFFFFF`) 아이콘**을 사용합니다. 이는 화려하고 다채로운 'Glass Gems & Orbs'와 시각적 대비를 이루어, 사용자가 데이터에 더 집중할 수 있도록 돕습니다.
*   **원칙:** 불필요한 장식을 배제하고, 사용자가 기능과 의미를 직관적으로 파악할 수 있도록 명료하게 디자인합니다.

## 6. UX/UI 아키텍처 (App Architecture)

앱은 하단의 네비게이션 바를 통해 4개의 핵심 페이지로 구성됩니다.

### **[ 하단 네비게이션 바 ]**
| 아이콘 | 페이지명 (영문) | 페이지명 (국문) |
| :---: | :--- | :--- |
| 🏠 | **Home** | **홈 (저널링)** |
| 💡 | **Insight** | **인사이트** |
| 🎯 | **Goal** | **목표** |
| 📊 | **Status** | **현황** |

---

### 6.1. Home (Journaling) 페이지
*   **키워드:** `기록`, `성찰` (Record, Reflection)
*   **사용자 질문:** **"오늘 내게 무슨 일이 있었고, 나는 무엇을 느꼈는가?"**
*   **핵심 기능:** 데이터 수집 및 당일 기록 시각화
*   **개발 및 UX 전략:**
    1.  **게이미피케이션:** 매일의 기록을 '보석(Gem)'으로 표현하고, 새로운 기록 수집 과정을 즐거운 경험으로 설계하여 동기를 부여합니다.
    2.  **부담 없는 인터페이스:** 틴더(Tinder)와 같은 스와이프 방식을 도입하여 사용자가 질문에 쉽고 빠르게 답변하며 감정을 기록할 수 있도록 설계합니다.
    3.  **시각적 피드백:** 기록이 완료되면 그날을 상징하는 독특한 'Gem'이 생성되는 것을 보여주어 즉각적인 성취감을 제공합니다.

### 6.2. Insight 페이지
*   **키워드:** `통찰`, `트렌드` (Insight, Trend)
*   **사용자 질문:** **"나는 무엇을 개선해야 할까?"**
*   **핵심 기능:** 일단위, 주단위, 월단위 데이터 기반 인사이트 제공
*   **개발 및 UX 전략:**
    1.  **사용자 최적화 시각화:** 수집된 데이터를 바탕으로 개인화된 'Orb'를 생성하여, 자신의 상태 변화 트렌드를 직관적으로 파악할 수 있게 합니다.
    2.  **예측 대시보드:** 미래의 마음/행동/신체 점수를 예측하여 보여줍니다.
        *   **UX Note:** 예측 점수는 확정적 수치가 아닌 **'잠재적 트렌드'**로 제시하여, 사용자가 가능성을 참고하되 불안감을 느끼지 않도록 주의합니다.
    3.  **스토리 형식의 리포트:** 다양한 분석 리포트를 제공하며, 클릭 시 인스타그램 스토리처럼 화면이 전환되어 지루하지 않게 정보를 탐색할 수 있습니다.

### 6.3. Goal 페이지
*   **키워드:** `계획`, `성장` (Plan, Progress)
*   **사용자 질문:** **"내가 되고 싶은 모습은 무엇이고, 얼마나 가까워졌나?"**
*   **핵심 기능:** 구체적인 행동 교정 및 성장 지원 프로그램 제안 및 관리
*   **개발 및 UX 전략:**
    1.  **프로그램 시각화:** 개별 프로그램의 성격과 유형을 각기 다른 'Gem' 형태로 시각화하여 사용자가 목표를 쉽게 식별하고 애착을 갖도록 합니다.
    2.  **진행 상황 시각화:** 목표 달성률, 진행 상황 등을 명확한 차트와 도표로 시각화하여 성취감을 고취시킵니다.
    3.  **검색 및 제안:** 사용자의 데이터와 목표를 기반으로 개인화된 프로그램을 추천하고, 원하는 목표를 직접 검색할 수 있는 기능을 제공합니다.

### 6.4. Status 페이지
*   **키워드:** `상태`, `성취` (State, Achievement)
*   **사용자 질문:** **"나는 지금까지 무엇을 해냈는가?"**
*   **핵심 기능:** 달성한 목표, 연속 기록, 주요 성과 요약
*   **개발 및 UX 전략:**
    1.  **핵심 지표 요약:** 총 기록 개수, 연속 기록 일수, 완료한 프로그램 수 등 핵심 성과 지표를 명확하게 보여줍니다.
    2.  **뱃지 시스템:** 의미 있는 목표를 달성하거나 특정 마일스톤에 도달했을 때 '뱃지'를 부여하여 성취를 기념하고 공유할 수 있도록 합니다.
    3.  **가치관 분석:** 축적된 데이터를 바탕으로 다른 사용자와의 가치관을 익명으로 비교 분석해주는 기능을 제공합니다.
        *   **UX Note:** 비교 분석은 순위나 경쟁을 유도하기보다, **자신의 고유한 특성을 발견하고 이해하는 데 도움을 주는 방향**으로 신중하게 설계합니다.
